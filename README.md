# Expense Tracker (Python)

A console-based Expense Tracker application built using Python to record, manage, and analyze daily expenses.  
The application supports CSV-based storage, category-wise summaries, and search functionality.

---

## Features
- Add expenses with date, amount, category, and optional notes
- View all stored expenses in a structured format                                                  
- Calculate total spending
- Generate category-wise spending summary
- Search expenses by category
- Persistent storage using CSV file handling
